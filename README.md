# Feature List
Here is a list of features/mechanics I want to make a prototype out of before production and networking:
* Breakable resources (stone or a tree)
  * You punch or mine the resource to harvest it
* Inventory Features
  * UI
  * Stacks for items, rather than one tile with a bunch of items
* Leveling
  * Experience system (gain xp by stuff like pve, harvesting, crafting)
  * Experience grants points to upgrade player
* Player Stats
  * Create stats for players
  > Will effect how certian interactions are done (more/less effectively)
* NPC System
  * Create basic npc interaction (inventory)
  * npc health, stam, ext
  * npc mounting (for creatures)
* PVP Systems
  * Create punch mechanic
  * Create weapons to be used in combat (guns, knives, ext)
* Building
  * Structure placing mechanic/snap grid (*Most important of the catagory*)
  * Defense structures (turrets, ext)
